# IS436 Project

## Project Description

This project is an IT Ticketing and Asset Management System created for Summit Ridge Clothing Company. The system was designed to improve how employees submit IT support requests and how the IT department manages tickets, assets, and software licenses.

Instead of using emails or informal communication, employees can use a centralized system to report issues, track ticket progress, and receive updates. IT staff can assign tickets, set priorities, and monitor company technology resources in one place.

## Use of Technology

This project uses multiple technologies to build, manage, and deploy the system:

- **Docker** was used to containerize the application, making it easier to run consistently across different environments.
- **GitHub** was used for version control, collaboration, and storing the project code.
- **Render** was used for web hosting and deployment so the project could be accessed online.
- **SQL** was used to design and manage the database for storing tickets, users, assets, and other system data.

These technologies work together to create a reliable, scalable, and organized solution for IT support management.

## Features

- Submit and track IT support tickets  
- Assign and prioritize requests  
- Manage hardware inventory  
- Track software licenses  
- Role-based user access  
- Online deployment for easy access
